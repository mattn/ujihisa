# ujihisa

便利

## Technique Description

This is clone of whitespace language. A space character is replaced to `便利`. A tab character is replaced to `感極まってきました`.

## Requirements

golang

## Installation

```
$ go get github.com/mattn/ujihisa/ujm
```

## Usage

This code should display `1`.

```
便利便利便利感極まってきました
感極まってきました
便利感極まってきました


```
